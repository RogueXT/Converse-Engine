# Converse-Engine

Pressure-Fed Cycle Based Liquid Rocket Engine integrated as thruster. 

**Propellants:**

Fuel : C3H8(L) 100%

Oxidizer: N20(L)

**Properties:**

Specific impulse (vac) : 268.45 s

Chamber Pressure : 700 psi
